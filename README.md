# ProjectSaveIt
A simple note API

Endpoints 

* Show All Notes : `GET api/notes/`
* Create Note : `POST api/notes/`
* Get Note by ID : `GET api/notes/:id`
* Update a Note : `PATCH api/notes/:id`
* Delete a Note : `DELETE api/notes/:id`
